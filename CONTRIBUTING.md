# Contributing to joinpizza.fun

Thanks for helping the Pizza & Friends Filipino tech community site.

## Report issues

Open a [GitHub issue](https://github.com/smmariquit/joinpizza.fun/issues) for broken links, event info, or copy updates.

## Developers

### Setup

```sh
bun install   # or npm install
bun dev       # http://localhost:4321
```

### Pull requests

1. Branch from **`staging`** (integration branch).
2. Use [Conventional Commits](https://www.conventionalcommits.org/) (`fix: …`, `content: …`).
3. Open a PR to **`staging`** with a short summary.
4. Do not commit secrets (`.env`, API keys).
